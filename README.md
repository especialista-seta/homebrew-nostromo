# Homebrew Tap for NOSTROMO

This is the official Homebrew tap for [NOSTROMO](https://github.com/especialista-seta/nostromo) - the MU-TH-UR 6000 AI Interface.

## Installation

```bash
brew tap especialista-seta/nostromo
brew install nostromo
```

## With Anthropic support

```bash
brew install nostromo
pip install anthropic  # Add Anthropic provider
```

## First-time Setup

```bash
nostromo configure
```

## Updating

```bash
brew update
brew upgrade nostromo
```

## Uninstall

```bash
brew uninstall nostromo
brew untap especialista-seta/nostromo
```
